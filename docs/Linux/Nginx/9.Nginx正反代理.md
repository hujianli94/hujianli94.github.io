## 正向代理


### Linux上临时配置正向代理

```shell
export proxy="154.213.3.219:80"
export http_proxy=$proxy
export https_proxy=$proxy
```


### Linux上永久配置正向代理

vim /etc/profile

```shell
export proxy="154.213.3.219:80"
export http_proxy=$proxy
export https_proxy=$proxy

# 使配置生效
source /etc/profile
```
