## Ventoy

一个制作可启动 U 盘的开源工具。无需反复地格式化 U 盘，只需要把 ISO 文件拷贝到 U 盘即可启动了，可一次性拷贝多个不同类型的 ISO 文件

### Github

- https://github.com/ventoy/Ventoy

- https://www.github-zh.com/projects/246335987-ventoy
