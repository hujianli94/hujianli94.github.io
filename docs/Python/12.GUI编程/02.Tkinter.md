### DouFTP 服务端

一个简单的 FTP 服务器端 GUI 应用程序，使用 Python 语言开发，GUI 基于 Tkinter，主要包含功能：

- 可以修改有权限的用户名、密码及端口
- 匿名用户访问只能浏览
- 可以修改路径，弹出文件夹选择窗口
- 启动和停止按钮可以交互显示
- 对文件夹路径做判断，不合理的会有提示

- https://douftp.org/server.html

源码地址：https://github.com/douftp/douftp-server
