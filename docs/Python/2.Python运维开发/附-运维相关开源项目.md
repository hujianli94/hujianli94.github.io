## 项目：lykchat+zabbix

实现微信告警

lykchat信息发送系统是Python3开发的，通过模拟微信网页端，基于个人微信号，为系统管理人员提供信息发送工具。

实现的功能有用户登录管理、微信登陆管理和微信信息发送功能。

地址：https://github.com/lykops/lykchat/
